# DeepDeen
A Islamic app DeepDeen is a professional Islamic PWA built with HTML, CSS, and JS. Features include Asma-ul-Husna, Asma-un-Nabi ﷺ, Quran (Read &amp; Audio), Hadith, Tasbeeh, Duas, Salah Times, and Hijri Calendar. It features a modern iOS-style Glassmorphism UI with optimized icon caching.
